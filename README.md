## Para analisar o projeto em sua máquina, siga este passo a passo: 😁

<br>

## 🇧🇷

### Clone o repositório😎

```
git clone https://github.com/jovimoura/fale-mais
```

### Acesse o diretorio🤓

```
cd <nome-da-pasta>
```
### Instale as dependências🤠
```
npm install
```
### Inicie a aplicação🤩
```
npm run dev
```
### A aplicação, por padrão, fica na porta:🤗

```
http://localhost:3000/
```
